## :hammer_and_wrench: Git Convention
1. Commit / Pull Request
    - `Feat` : 새로운 기능 추가
    - `Fix` : 오류 및 버그 수정
    - `Docs` : 문서 (README, 메뉴얼 등)
    - `Test` : 테스트용 코드
    - `Chore` : 빌드, 설정 파일
    - `Comment` : 주석 추가
2. Git Branch Naming
    - `main` : 기본 Branch
    - `docs` : 문서 작업 Branch
    - `Assn[Assn_num]` : 각 과제를 작업하는 Branch
